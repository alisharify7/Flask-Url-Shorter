# Url-Shorter
a Flask Web Application For  create Short links


## For Run

- Windows :

      python -m venv venv
      .\venv\Scripts\activate
      
      
- linux or mac :
  
      python3 -m venv venv
      source ./venv/bin/activate


then Just type:
            
      flask run
      or
      Python app.py
