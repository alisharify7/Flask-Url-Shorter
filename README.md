# Url-Shorter
a Flask Web Application For  create Short links


## For Run

- Windows :

      python -m venv venv
      .\venv\Scripts\activate
      
      
- linux or mac :
  
      python3 -m venv venv
      source ./venv/bin/activate


then Just type:
            
      flask run
      or
      Python app.py


### >by default this web App set on 127.0.0.1:8080 You can change it from config file
